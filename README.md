## Install

Clone repo

```
git clone https://github.com/OmaRmz/netcommerce.git
```

## How to setting 

Run the migration

```
php artisan migrate
```

Or run the migration with seeder if you want seeding the related data

```
php artisan migrate --seed
```

Generate a New Application Key

```
php artisan key:generate
```
